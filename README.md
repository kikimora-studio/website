# Kikimora Studio

Website for Kikimora Studio - a fitness and wellness studio featuring Reformer Pilates and Bodyshape V2 technology.

## Tech Stack

- **Astro** - Static site generator
- **TailwindCSS** - Utility-first CSS
- **Alpine.js** - Lightweight interactivity
- **GLightbox** - Image lightbox

## Features

- Bilingual support (Bulgarian / English)
- Dark mode
- Responsive design
- Equipment showcase
- Mobile app preview
- Gallery with lightbox

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
